# Current Visual Upgrade Notes

This repo now includes a better placeholder art pass intended to move the prototype closer to Porto without shipping external licensed content yet.

## What Changed In-Code

- Replaced the generic skyline with layered Porto-inspired waterfront silhouettes.
- Added Ribeira-like colorful facades with azulejo accents, balconies, rooflines, and better height variation.
- Added a stronger Dom Luís bridge silhouette and steel truss rhythm.
- Added quay walls, river haze, rabelo boats, lamp posts, and more grounded reflections.
- Improved the placeholder character textures for Herosauro, Super Boxy, golems, and Adamastor so they read more like a production target than debug icons.

## Why This Is Still A Placeholder Pass

- All current visual upgrades are procedural and license-safe.
- They are meant to guide art direction, not replace final art production.
- The next serious step should be a 3D-to-2D render pipeline for heroes and backgrounds.

## Best Next Art Task

Replace the current Stage 1 backdrop with one authored background package:

1. far skyline matte
2. Ribeira facade layer
3. bridge layer
4. quay layer
5. water reflection layer
6. foreground prop set

That single package would do more for perceived production value than spreading effort across all stages too early.
